## Notes
- Authentication uses JWT tokens stored in localStorage.
- Real-time stock prices are received via WebSocket at `/ws`.
- All prices and numeric values should use tabular lining for visual consistency.
- The UI follows a clean, minimal, modern trading aesthetic similar to professional platforms.
